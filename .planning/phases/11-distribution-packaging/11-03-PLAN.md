- `source: "./"` = repo root (marketplace root = directory containing `.claude-plugin/`)
- `strict: true` (default) → plugin.json is authority, marketplace entry supplements
- NO `version` field in marketplace.json (explicit docs warning: "plugin manifest always wins silently")
