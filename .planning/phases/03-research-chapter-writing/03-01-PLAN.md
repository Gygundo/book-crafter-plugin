**Section 5: Scripture Accuracy Rules (RSRCH-02)**
Include these rules prominently in the skill:
- Quote actual NKJV text. Include full verses, not fragments.
- If uncertain about the exact wording of a passage, write the reference and add `<!-- VERIFY -->` after the quoted text
- Never paraphrase scripture and present it as a direct quote
- Never invent or fabricate scripture references
- Favour well-known passages (Genesis, Psalms, Proverbs, Matthew, John, Romans, Ephesians, Hebrews) where they fit the argument
- For less-common references, add `<!-- VERIFY -->` as a safety net
- Do not use "The Bible says..." without a specific reference

**Section 6: Output Rules (RSRCH-03)**
- Output path: `[project_directory]/research/ch[NN]-research.md` where NN is zero-padded (ch01, ch02, etc.)
- Create the research/ directory if it does not exist
- Write the complete artefact to the file
- Return a brief summary to the orchestrator: "Research complete for Chapter [N]: [Title]. [X] primary scriptures, [Y] word studies, [Z] cross-references."

**Section 7: Depth Calibration (RSRCH-04)**
- Minimum per chapter: 2 primary scripture passages, 1 word study, 2 cross-references, 1 type/shadow, 2 illustrations
- For Climax momentum chapters: increase minimums to 3 primary passages, 2 word studies, 3 cross-references
- For Foundation chapters: include extra illustrations (3 instead of 2) to build reader trust
- Research artefact should be 800-1500 words (dense, not verbose -- the writer will expand)
